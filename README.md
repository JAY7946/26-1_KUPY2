# AI-based RTL PPA Predictor

Deep learning based predictor for semiconductor RTL design metrics.

## Input
- Gate Count
- Wire Length
- Fanout
- Logic Depth
- Frequency
- VDD
- Toggle Rate
- Temperature

## Output
- Power
- Delay
- Area

## Deployment
Streamlit web app

## Model
PyTorch DNN
